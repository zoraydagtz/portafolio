MIXA | Artesanías

Plataforma E-Commerce de Comercio Justo

Contexto del Proyecto

MIXA es una plataforma e-commerce B2C diseñada para conectar a artesanos mexicanos de los 32 estados directamente con consumidores que valoran lo auténtico. La plataforma elimina intermediarios abusivos y promueve el comercio justo, permitiendo a los artesanos fijar sus propios precios y recibir una retribución digna.

Mi Rol

Frontend Developer & Coordinadora Técnica

  • Desarrollo de interfaces responsivas con HTML5, CSS3 y JavaScript
  • Implementación de Bootstrap para diseño mobile-first
  • Creación de user stories siguiendo Definition of Ready
  • Coordinación de Git workflows y resolución de merge conflicts
  • Diseño de wireframes en Figma con paleta oficial
  • Validación de formularios con expresiones regulares
Tecnologías Utilizadas
Java Spring Boot MySQL HTML5 CSS3 JavaScript Bootstrap Git/GitHub SCRUM Jira Figma
Desafíos Enfrentados

Coordinación de equipo grande (10 personas):

Implementamos SCRUM con sprints de una semana, daily standups y retrospectivas. Usamos Jira para tracking y Git branches para desarrollo paralelo sin conflictos.

Diseño inclusivo para artesanos con poca experiencia digital:

Creamos una interfaz intuitiva con navegación clara, formularios simples y feedback visual constante para facilitar la carga de productos.

Bootcamp Project

Tienda Deportiva

E-commerce de Productos Deportivos

Descripción

Sistema de comercio electrónico especializado en productos deportivos. La plataforma incluye catálogo de productos, sistema de filtrado por categorías, carrito de compras funcional y gestión de inventario.

Características Principales
  • Catálogo dinámico de productos con imágenes y especificaciones
  • Sistema de filtrado por categoría, precio y marca
  • Carrito de compras con gestión de cantidades
  • Diseño responsivo con CSS3 y media queries
  • Validación de formularios con JavaScript
  • Interfaz de usuario intuitiva y moderna
Stack Técnico
HTML5 CSS3 JavaScript Git/GitHub
Aprendizajes Clave

Este proyecto me permitió profundizar en manipulación del DOM, eventos en JavaScript, y diseño responsivo. También practiqué el flujo de trabajo con Git, creando branches para features y manejando pull requests.
